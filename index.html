Volunteer Hub
 Coordinate volunteers and find community projects to get involved with in your area.
 Volunteer Hub allows organisers of volunteer driven projects to attract and organise people; they can be arranged into groups according to their specified skill sets and schedule availabilty using the inbuilt tools.
 These groups can then be organised using the group-chat feature or by posting to the community noticeboard.

Popquest
 Create and find routes to discover new places.
 Routes can be designed as a simple walking tour with descriptions or as an interactive game in the style of a treasure hunt, with the ability to trigger the next stop with a location based question or a proximity check that only shows the next stop when the player has successfully found the right location using the provided clues.

Tetris
 A recreation of the classic game using core web technologies.
 A 1 week solo project where the brief was to reproduce a grid-based game in the browser. This involved creating a simple engine in JavaScript that was capable of manipulating two-dimensional arrays and keeping track of the state of various objects.
 This project was heavily reliant on smooth user-experience and polished presentation.

NewsFeed
 A 2 day hackathon project for creating a front end to consume the News API, presenting the information in a clean and readable format with a vibrant style and simple animations to encourage user engagement.
 Aggregates the news from around the world and allows the user to search by keyword, source or country. From the results, the user can build a customised new feed that will populate the front page upon revisiting.
